# NingboMedicalInsuranceShop
宁波医保零售点位置

# 项目原因

医院门诊没有需要开给我的一个处方药，于是医生开了处方单，让我自己去外面买。

结果，完全不知道哪些药店是有医保的，并且在附近的；找到了可用医保的药店，却没有需要的药。

于是，写了那么一个东西，从官方的渠道获得所有宁波能够使用医保的药店，然后放入地图，这样就知道附近有哪些医保的药店了。

不至于茫无目的的去找药店，而是直接去找医保的药店，这样就可以节省时间。

# 数据源

官方没有搜索功能，或者说搜索功能完全不起作用。

读取数据非常慢，代码中从数据库一次性（一个http request）取100条数据，500M网络带宽个位数毫秒延迟情况下，需要20秒左右的时间。
